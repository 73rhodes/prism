How to use

This plugin is based on the line-highlight plugin and only works on code blocks (<pre><code>). It allows you highlight code sections with different colors and attach an optional text annotation tooltip.

Known issues: this plugin is known not to work with the COY theme due to background scrolling issues.

Specify the lines to be highlighted using the data-coverage attribute on the <pre> element. The format is described below.

  • The basic format is class:range|text,range|text;...
  • The class can be one of line-covered, line-uncovered, or a custom CSS class.

A range is specified as follows:

  • A single number refers to the line with that number
  • Ranges are denoted by two numbers, separated with a hyphen (-)
  • Multiple line numbers or ranges are separated by commas.
  • Whitespace is allowed anywhere and will be stripped off.
  • An optional tooltip message can be added as |some text at the end of a range.

Examples:

line-covered:5
Apply the line-covered style to the 5th line
1-5
Lines 1 through 5
1,4
Line 1 and line 4
1-2, 5, 9-20
Lines 1 through 2, line 5, lines 9 through 20

The same data-line-offset attribute used in the line-highlight plugin is supported.

You can also link to specific lines on any code snippet, by using the following as a url hash: #{element-id}.{lines} where {element-id} is the id of the <pre> element and {lines} is one or more lines or line ranges that follow the format outlined above. For example, if there is an element with id="play" on the page, you can link to lines 5-6 by linking to #play.3-12
